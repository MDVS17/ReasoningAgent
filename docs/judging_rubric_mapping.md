# Syntrix Judging Rubric Mapping

## Accuracy & Relevance

Syntrix recommendations are tied to synthetic Copilot-style work interactions, repeated workflow patterns, and transparent opportunity scoring. The generated blueprint maps back to triggering work patterns, target user, guardrails, evaluation tests, and cited IQ evidence.

## Reasoning & Multi-Step Thinking

Syntrix uses a Master Agent plus specialist agents:

- Signal Discovery Agent
- Pattern Scoring Agent
- Blueprint Architect Agent
- Safety Governance Agent
- Learning Loop Agent

`POST /api/analyze` returns a reasoning trace that shows each handoff, decision, and output.

## Creativity & Originality

Most agent builders ask users to configure agents manually. Syntrix reverses the workflow: it discovers agent opportunities from repeated work signals, then generates safe personalized agent blueprints. The product thesis is memorable and enterprise-relevant.

## User Experience & Presentation

The FastAPI demo serves a cinematic frontend with a clear story:

- Hero and problem narrative
- Workspace Signal Snapshot
- Task frequency and Opportunity Score charts
- Interactive multi-agent system
- Reasoning trace
- Generated Agent Blueprint
- Microsoft IQ evidence panel
- Governance Gates
- Syntrix Learning Loop

## Reliability & Safety

Syntrix runs locally with deterministic logic, synthetic data, no secrets, and no paid APIs. Governance gates require human approval before external communication, system changes, sensitive content handling, or write-capable actions.

## Community Vote

Syntrix is easy to explain in one sentence: from scattered Copilot usage to personalized AI agents. The product has a clear enterprise use case, polished demo narrative, and practical safety stance.
