# Safe Agent Deployment Checklist

## Purpose

This synthetic source provides a checklist for safe local demo blueprint review.

## Checklist

- Confirm the agent uses approved sources.
- Confirm the agent has a clear target user and workflow.
- Confirm sensitive content is flagged.
- Confirm external communication requires human approval.
- Confirm system changes require human approval.
- Confirm evaluation tests cover quality, safety, and traceability.

## Deployment Rule

The local demo recommends blueprint updates only. It does not silently retrain a model or perform autonomous write actions.
