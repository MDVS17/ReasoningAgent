# Blueprint Quality Standards

## Purpose

This synthetic source defines quality standards for generated Syntrix Agent Blueprints.

## Required Sections

- Agent name
- Purpose
- Target user
- Triggering work patterns
- Capabilities
- Required knowledge sources
- Guardrails
- Human approval gates
- Evaluation tests
- Continuous improvement path

## Quality Bar

The blueprint should be specific, explainable, grounded in repeated work signals, and safe enough for human review.
